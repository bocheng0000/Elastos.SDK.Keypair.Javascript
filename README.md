## Install

`yarn add elastos-wallet-js`
or
`npm install elastos-wallet-js`

## Development

* `yarn` or `npm install` to install dependencies
* `yarn test` to run tests

## Docs

https://elastos-wallet-lib-javascript.readthedocs.io/
